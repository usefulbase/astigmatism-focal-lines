# 焦線と網膜

眼鏡店スタッフ向けに、乱視における焦線と網膜の位置関係を視覚的に学ぶための教育支援ツールです。

図上で前焦線・後焦線・最小錯乱円の位置を確認しながら、目の状態（S/C/AX）や補正レンズの変化を操作できます。

## できること

- 網膜に対する水平・垂直焦線の位置を図で確認
- 焦線間の中点として最小錯乱円を表示
- 現在の目の状態を `S / C / AX（180°・90°）` で設定
- 度数転換と正視へのリセット
- 補正レンズ（球面・円柱）の手動調整
- クロスシリンダーの1番・2番の見え方と、焦線移動のイメージ確認
- 必要なときだけ「焦線を直接調整」を開く操作
- 図の拡大、PC・スマートフォンの縦横表示への対応
- PWA対応：ホーム画面に追加してアプリのように起動、初回表示後のオフライン利用

## 使い方

1. まず「現在の目の状態」で S・C・AX を設定します。
2. 図の焦線と網膜（眼球後方の実線）の関係を確認します。
3. 補正の練習では「補正レンズシミュレーションを表示」を開き、S / C を手動で調整します。
4. クロスシリンダーを使う場合は、1番（白）・2番（赤）のヒントを参考に C の方向を判断します。
5. 図を直接動かして理解したい場合だけ、「焦線を直接調整」を開きます。

補正軸（AX）は決定済みという前提で、クロスシリンダーの練習では軸を動かさず円柱度数（C）の比較・調整を行います。

## スマホでアプリとして使う

公開ページを開き、ホーム画面に追加するとアプリのように起動できます。

- iPhone / iPad：Safari の共有ボタンから **ホーム画面に追加**
- Android：Chrome のメニューから **アプリをインストール** または **ホーム画面に追加**

初回は通信できる状態でページを開いてください。その後は、以前開いた内容をオフラインでも起動できます。

## 注意

本ツールは焦線とレンズ補正の関係を理解するための教育用イメージです。実際の屈折検査・処方判断・診断の代替には使用しません。
